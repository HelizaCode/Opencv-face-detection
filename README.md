# 🎥 Real-Time Face Detection using OpenCV

## 📌 Overview

This project demonstrates real-time face detection using a webcam with OpenCV in Python. It uses a Haar Cascade classifier to detect human faces in video frames.

## 🧠 Technologies Used

* Python
* OpenCV
* Haar Cascade Classifier

## 📂 Project Structure

```
face_detection.py
haarcascade_frontalface_default.xml
requirements.txt
```

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/opencv-face-detection.git
cd opencv-face-detection
```

2. Install dependencies:

```
pip install -r requirements.txt
```

## ▶️ Usage

Run the script:

```
python face_detection.py
```

Press **ESC** to exit.

## 📷 Output

* Detects faces in real-time
* Draws a rectangle around detected faces

## 📌 Notes

* Make sure your webcam is connected
* Haar Cascade XML file must be in the same directory
